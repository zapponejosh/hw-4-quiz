# JavaScript Quiz

A JS quiz!  10 multiple choice questions. 60 seconds to complete. A list of high scores that persists in localStorage

_Do not use this as a secure way to generate a password._

#### Uses:
* CSS Grid
* jQuery
* event listeners
* localStorage
* And much more!
